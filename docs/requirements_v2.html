ダッシュボード ニュース 公式ソース

要件定義 v2

KPIダッシュボード + ニュース | 静的配信 + 日次更新 | JSON-LD | A11y

目的

  • 転職市場の主要KPIとニュースをワンストップで提供し、意思決定を支援する。
  • 出典・更新時刻を明示し、信頼性と再現性を担保する。

対象ユーザー

  • 求職者・キャリア支援者・採用担当者
  • 業界アナリスト・人材関連メディア

画面

ダッシュボード（index.html）

  • 有効求人倍率の推移（移動平均、注釈、前月比/前年比バッジ）
  • 産業別 平均年収（平均/中央値ライン）
  • 地域別 求人シェア（上位選択、トップ表示）
  • 年代別 転職成功率（チェック式比較、統計バッジ）
  • 最終更新と出典の表示

ニュース（news.html）

  • カテゴリ/タグでフィルタ、モーダルで本文表示
  • 構造化データ: JSON-LD（NewsArticle）
  • 出典URL、組織名、更新頻度を明示

データ仕様

  • data/kpi.json: メトリクス（jobRatio, salaryByIndustry, regionJobShare, successRateByAge）、lastUpdated、sources
  • data/news.json: items[]（id, title, category, date, fullContent, sourceUrl 他）、lastUpdated
  • 取得/生成: scripts/fetch_news.mjs（日次）/scripts/import_text_to_news.mjs（手動）

非機能要件

  • 配信: GitHub Pages（静的）
  • 更新: GitHub Actions（毎日 03:00 UTC ≈ 12:00 JST）で data/news.json をコミット
  • 互換: 最新版 Chrome/Edge/Safari、横幅360px以上で閲覧可能
  • パフォーマンス: 初回表示で主要コンテンツが1.5s以内（ローカル測定目安）

SEO / 構造化データ

  • 各記事に schema.org NewsArticle（JSON-LD）を埋め込み
  • OGP/メタ: タイトル、説明、OG画像（共通）
  • サイトマップ/RSSは将来対応（任意）

UX / アクセシビリティ

  • モーダルのフォーカストラップ、Escで閉じる、背景スクロール抑制
  • キーボード操作（Tab/Shift+Tab）で主要UIにアクセス可
  • カラーコントラストはWCAG AA相当を目安
  • レスポンシブ（PC/タブレット/スマホ）

受入基準

  • コンソールエラーがない
  • ダッシュボードの各チャートとバッジが表示される
  • data/*.json の日付が最新で、出典リンクが有効
  • ニュース詳細モーダルの本文・出典・JSON-LDが正しく生成される

将来拡張（任意）

  • 比較・料金ページ（/compare, /pricing）の追加
  • より多様な公式フィード対応（e-Stat 等）
  • テスト導入（Playwright など）
