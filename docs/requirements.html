転職者向け情報サイト - 要件定義書

プロジェクト概要

エグゼクティブサマリー

転職者向け情報サイト（career-horizon.com準拠）として、転職市場のKPI指標をリアルタイムで可視化するダッシュボードページと、転職に関する最新ニュースを掲載するニュースページの2つの主要コンテンツで構成されるシンプルな情報サイトを構築します。

サイト構成
2ページ
ダッシュボード + ニュース

主要コンテンツ1: KPI指標ダッシュボード

転職市場の動向、求人倍率、平均年収、業界別転職率などを視覚的に表示するダッシュボード

主要コンテンツ2: 転職ニュースページ

転職市場の最新ニュース、業界動向、転職成功事例、専門家コラムを掲載するニュースページ

UIデザイン案

案1: ダッシュボード特化型

特徴: データ可視化に特化したビジネス・アナリティクス系デザイン

採用理由: KPI指標を効果的に表示するため、グラフやチャートが見やすい構成

画像生成プロンプト: "Modern analytics dashboard UI, career KPI metrics, charts and graphs, dark blue theme, professional data visualization, clean layout with sidebar navigation"

案2: ニュース・メディア型

特徴: ニュースサイト風のクリーンで読みやすいレイアウト

採用理由: ニュース記事の閲覧性を重視し、記事一覧とダッシュボードを両立

画像生成プロンプト: "Clean news website UI, article cards layout, dashboard widgets integration, modern typography, white and blue color scheme, responsive design"

機能要件

サイト構成図

KPI指標詳細

表示指標

  • 求人倍率: 月次・年次推移グラフ
  • 平均年収: 業界別・職種別比較チャート
  • 転職成功率: 年代別・経験年数別データ
  • 求人数推移: 地域別・時系列データ

ユーザーストーリー

「転職を検討している私は、現在の転職市場の状況を把握したいので、KPI指標をダッシュボードで確認し、最新の業界ニュースを読みたい」

非機能要件

パフォーマンス要件

  • ページ表示速度: 3秒以内
  • 同時接続数: 10,000ユーザー
  • 検索レスポンス: 1秒以内

セキュリティ要件

  • SSL/TLS暗号化通信
  • 個人情報保護法準拠
  • GDPR対応
  • 2段階認証対応

システムアーキテクチャ

graph TB A[ユーザー] --> B[Webブラウザ] B --> C[静的サイト/CDN] C --> D[ダッシュボードページ] C --> E[ニュースページ] F[データ収集API] --> G[KPI指標DB] H[ニュースCMS] --> I[記事DB] D --> F E --> H J[管理者] --> H K[外部データソース] --> F

技術スタック（シンプル構成）

  • フロントエンド: HTML, CSS, JavaScript（Chart.js等）
  • バックエンド: Node.js または Python（データ取得）
  • データベース: JSON/CSV形式またはMySQL（軽量）
  • ホスティング: Netlify, Vercel, GitHub Pages
  • データ可視化: Chart.js, D3.js
