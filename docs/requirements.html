要件定義 v1 アーカイブ

最終更新: 2024-08-12 / 初期ローンチ用に作成した最小構成の要件メモ。最新版は 要件定義 v2 を参照してください。

プロダクト概要

  • 目的: 転職市場の主要 KPI と最新ニュースを 1 つのサイトで提供し、キャリア意思決定を支援する。
  • 対象: 首都圏のビジネス職（マーケ/セールス/PM）およびキャリアアドバイザー。
  • 範囲: 静的サイト（Jamstack）での公開。ニュースは JSON で収集し、ダッシュボードは Chart.js で可視化。

主要機能

  1. KPI ダッシュボード: 求人倍率、給与推移、地域別求人シェア、年代別成功率。
  2. ニュース一覧: カテゴリ・タグでフィルタ可能。モーダルで本文表示。
  3. リソース集: 履歴書テンプレート、交渉スクリプトのダウンロード。
  4. 用語集: KPI や採用用語を簡易的に説明。

データ仕様

  • data/kpi.json: KPI 数値と更新日を保持。月次更新。
  • data/news.json: ニュースのタイトル、本文、カテゴリを保持。日次更新。
  • データソース: 厚生労働省、総務省統計局、求人メディア各社の公開情報。

制約事項

  • CMS なし。手動で JSON を更新。
  • ダークモードやアクセシビリティは最小限（コントラストのみ）。
  • ニュース要約やインサイトの付加価値は未実装。

移行メモ

v2 では以下を改善:

  • フィルタリングとパーソナライズ機能の強化。
  • リソースやツールの充実（年収シミュレーター、スキル診断）。
  • アクセシビリティとレスポンシブ強化。
  • コンテンツの深さ（ニュース要約、企業比較機能）。
